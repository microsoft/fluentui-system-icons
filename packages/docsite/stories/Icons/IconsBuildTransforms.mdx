<h1 className="sbdocs-title">Build-Time Import Transforms</h1>

You can keep root-level barrel imports and leverage build-time transforms to automatically rewrite them to optimized atomic imports — no source code changes required.

These transforms rewrite imports from:

```tsx
import { AccessTime24Filled } from '@fluentui/react-icons';
```

To the optimized atomic import for your chosen target:

```tsx
// SVG (standard)
import { AccessTime24Filled } from '@fluentui/react-icons/svg/access-time';

// SVG sprites (⚠️ preview)
import { AccessTime24Filled } from '@fluentui/react-icons/svg-sprite/access-time';

// Font icons
import { AccessTime24Filled } from '@fluentui/react-icons/fonts/access-time';

// Headless API (SVG)
import { AccessTime24Filled } from '@fluentui/react-icons/headless/svg/access-time';

// Headless API (fonts)
import { AccessTime24Filled } from '@fluentui/react-icons/headless/fonts/access-time';
```

The examples below use `svg` as the target path. Replace it with the appropriate path for your setup from the list above.

## Webpack (recommended)

We ship two dedicated webpack packages that, used together, give you the smallest possible bundles with zero source changes:

1. **[@fluentui/react-icons-atomic-webpack-loader](https://www.npmjs.com/package/@fluentui/react-icons-atomic-webpack-loader)** — rewrites barrel imports into atomic deep paths for optimal tree-shaking.
2. **[@fluentui/react-icons-font-subsetting-webpack-plugin](https://www.npmjs.com/package/@fluentui/react-icons-font-subsetting-webpack-plugin)** — when using the font variant, subsets the icon font files down to only the glyphs your app actually uses.

### 1. Atomic import loader

Install the loader:

```bash
npm install --save-dev @fluentui/react-icons-atomic-webpack-loader
```

Add it as an `enforce: 'pre'` rule so it runs **before** other loaders:

```js
// @filename webpack.config.js
module.exports = {
  module: {
    rules: [
      {
        test: /\.[mc]?[jt]sx?$/,
        enforce: 'pre',
        use: ['@fluentui/react-icons-atomic-webpack-loader'],
      },
      // … your other rules (babel-loader, ts-loader, etc.)
    ],
  },
};
```

**Important:** Do **not** exclude `node_modules` — the loader must be able to transform barrel imports found inside dependencies too. Files without `@fluentui/react-icons` references are skipped via a fast regex pre-check, so there is no meaningful overhead.

Pass the `iconVariant` option to target a different implementation (`'svg'` — default, `'svg-sprite'`, or `'fonts'`). To adopt the Headless API, add `headless: true`:

```js
{
  test: /\.[mc]?[jt]sx?$/,
  enforce: 'pre',
  use: [
    {
      loader: '@fluentui/react-icons-atomic-webpack-loader',
      // e.g. headless font icons
      options: { iconVariant: 'fonts', headless: true },
    },
  ],
}
```

### 2. Font subsetting plugin

When you use the font variant (`iconVariant: 'fonts'`), pair the loader with the font subsetting plugin so only the glyphs used by your build are shipped:

```bash
npm install --save-dev @fluentui/react-icons-font-subsetting-webpack-plugin
```

```js
// @filename webpack.config.js
const {
  default: FluentUIReactIconsFontSubsettingPlugin,
} = require('@fluentui/react-icons-font-subsetting-webpack-plugin');

module.exports = {
  module: {
    rules: [
      // The atomic loader from step 1
      {
        test: /\.[mc]?[jt]sx?$/,
        enforce: 'pre',
        use: [
          {
            loader: '@fluentui/react-icons-atomic-webpack-loader',
            options: { iconVariant: 'fonts' },
          },
        ],
      },
      // Treat the font files as webpack assets so they can be subset
      {
        test: /\.(ttf|woff2?)$/,
        type: 'asset',
      },
    ],
  },
  plugins: [
    // Subsets font files to only the glyphs used by your build
    new FluentUIReactIconsFontSubsettingPlugin(),
  ],
};
```

> **Note:** Subsetting requires `optimization.usedExports` to be enabled, which webpack does by default in production mode. Without this plugin the full font files (all icon glyphs) are bundled, which is **not recommended**.
